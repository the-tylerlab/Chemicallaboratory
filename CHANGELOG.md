# Changelog (บันทึกการปรับปรุงระบบ)

บันทึกประวัติการพัฒนาและการอัปเกรดระบบจัดการห้องปฏิบัติการเคมีและอุปกรณ์วิทยาศาสตร์ (Chemical Laboratory & Scientific Equipment System)

## [1.9.1] - 2026-07-01
### Added
- **ระบบพิมพ์สติกเกอร์บาร์โค้ด QR Code แบบแอนิเมชัน (Animated QR Code Label Printer):**
  - เพิ่มป๊อปอัพจำลองเครื่องพิมพ์สติกเกอร์ฉลากติดขวดสารเคมีและอุปกรณ์ (Label Printer Overlay) เมื่อกดสั่งพิมพ์
  - จำลองตัวเครื่องพิมพ์แบบ 3D Thermal Printer และป้ายสติกเกอร์ที่ค่อยๆ เลื่อนเลื่อนตัวออกมา (Sliding Sticker Roll out)
  - ติดตั้งเสียงพิมพ์แบบ Stepper/Thermal Buzz ด้วย Web Audio API ที่ตรงตามจังหวะอนิเมชันกระดาษเลื่อน
  - แสดงฉลากสติกเกอร์ที่มีรหัส QR Code สำหรับสแกน, ชื่อสารเคมี/อุปกรณ์, รหัสโค้ดของไอเทม และสถานที่จัดเก็บตู้/ชั้นวางในห้องแล็บที่ถูกต้อง พร้อมปุ่มกดสั่งพิมพ์ฉลากทางกายภาพ (Print Label)

## [1.9.0] - 2026-06-26
### Added
- **ระบบล้างข้อมูลของแอดมิน (Admin Data Clearing Feature):**
  - เพิ่มปุ่ม "ล้างข้อมูล" แยกในแต่ละโมดูล ได้แก่ คลังพัสดุ สารเคมี ประวัติการยืม-คืน การจองห้องปฏิบัติการ รายการสั่งซื้อสะสม และแผนการทดลอง
  - ปุ่มล้างข้อมูลจะแสดงเฉพาะสำหรับผู้เข้าสู่ระบบในฐานะแอดมิน (`userRole === "admin"`)
  - นำเสนอหน้าต่างการยืนยันการลบที่ปลอดภัย โดยบังคับให้พิมพ์ประโยค `CONFIRM TO DELETE` (ตัวพิมพ์ใหญ่ทั้งหมด) เพื่อปลดล็อคปุ่มยืนยัน
  - รองรับการล้างข้อมูลแบบ Synchronous/Asynchronous ทั้งในระดับ Local Storage, Backend API และ Firebase Firestore (ลบแบบ doc-by-doc)
- **แอนิเมชันจำลองเครื่องพิมพ์ใบยืนยันการจอง (Booking Receipt Printer Animation & Sound - ปิดใช้งานชั่วคราวตามคำขอ):**
  - พัฒนาโครงสร้างการจำลองช่องเสียบเครื่องพิมพ์ (Printer Slot) และตัวกระดาษข้อมูลแบบรอยฉีกหยัก (Jagged/Zigzag cut paper) บนป๊อปอัพยืนยันจองห้องสำเร็จ
  - สร้างแอนิเมชันกระดาษเลื่อนปริ้นท์ลงมาจากเครื่องตามจังหวะจริง
  - ผสมผสานเสียงจำลองการพิมพ์ด้วยเครื่องจักรความถี่สูง (Mechanical Print Buzz) ด้วย Web Audio API เพื่อให้ได้เอฟเฟคเสียงประสานที่ตื่นตาตื่นใจและพรีเมียมโดยไม่ต้องดาวน์โหลดไฟล์เสียงภายนอก
- **เครื่องมือวิทยาศาสตร์แสนสนุกสำหรับนักเรียน (Student-Friendly Science Tools Web App - เอาออกชั่วคราวตามคำขอ):**
  - พัฒนาหน้าเว็บแอปพลิเคชันแบบ SPA (Single-Page Application) ในดีไซน์สีสันพาสเทลสดใสและเป็นมิตรกับนักเรียนระดับมัธยม
  - **เคมีวิเศษ (Magic Periodic Table):** ตารางธาตุจำลองแสดงธาตุ 20 ลำดับแรกและธาตุหลักในชีวิตประจำวัน พร้อมจำลองภาพการหมุนของระดับชั้นพลังงานอิเล็กตรอน (Bohr Model) แบบเรียลไทม์ผ่าน Canvas
  - **นักสืบยีนส์เด่น-ยีนส์ด้อย (Genetics Punnett Square Simulator):** เครื่องมือผสมคู่ยีนส์วิเคราะห์ตามกฎของเมนเดล แสดงอัตราส่วนสัดส่วนของโอกาส พร้อมภาพจำลองลักษณะภายนอก (Phenotype) ของดอกไม้และสีตาผ่าน SVG
  - **แปลงกายฟิสิกส์ (Physics Calculators & Converter):** ตัวแปลงหน่วยพื้นฐาน และเครื่องจำลองเคลื่อนไหวตามสูตรความเร็ว (มีรถอนิเมชันวิ่งตามค่าจริง), แรงผลักดัน และการจม-ลอยของวัตถุตามสัดส่วนความหนาแน่นในน้ำ
  - **พลอตเตอร์สีรุ้ง (Rainbow Graph Plotter):** เครื่องมือป้อนข้อมูลพิกัด X-Y เพื่อแสดงกราฟวิเคราะห์สำหรับวิชาวิทยาศาสตร์ โดยเลือกสัญลักษณ์จุดพิกัดเป็นรูปทรงแสนน่ารัก เช่น "ดาว" หรือ "หัวใจ" พร้อมคำนวณและวาดเส้นแนวโน้มอัตโนมัติ (Linear Regression Line)
  - เชื่อมต่อลิงก์เปิดหน้าต่างใหม่ใน Sidebar ของแอปพลิเคชันหลักโดยตกแต่งสีสะดุดตาโดดเด่นเพื่อนักเรียนเข้าใช้งานได้สะดวก

### Changed
- **ปรับปรุงดีไซน์ Sidebar Bottom Menu & Footer:**
  - นำเส้นแบ่งขอบบน (Border lines) ทั้งสองเส้นในส่วนล่างของเมนูด้านข้างออกทั้งหมดตามความต้องการของผู้ใช้ เพื่อความสะอาด เรียบง่าย และสบายตาตามแนวทางมินิมอล
- **เพิ่มรายละเอียดสรุปยอดราคารวมในรายการสั่งซื้อสะสม (Committed PO Summary Details):**
  - ปรับปรุงแถบสรุปท้ายตารางสั่งซื้อพัสดุให้แสดงราคารายละเอียดครบถ้วน: ราคารวมก่อนลดราคา, เปอร์เซ็นต์ส่วนลดสะสมโดยเฉลี่ย, มูลค่าส่วนลดสะสม และราคาหลังจากหักส่วนลดสุทธิทั้งหมด ปรับสไตล์ให้สวยงามชัดเจนและมีระเบียบยิ่งขึ้น
- **อัปเกรดความสามารถระบบแชทบอตอัจฉริยะ (Smart Chatbot - Catalyst Upgrades):**
  - **แก้ไขปัญหาทับซ้อนของการจับคู่คำ (Overlap Prevention & Word Boundaries):** แก้ไขบั๊กการจับคู่คำค้นหาสูตรเคมีทับซ้อน (เช่น การจับคำว่า `"na"` แยกออกมาเป็นสารเคมี "โซเดียม" ซ้อนทับอยู่ภายใต้สูตรของ `"naoh"` ทำให้แชทบอตเข้าใจผิดว่าเป็นปฏิกิริยาระหว่าง NaOH และ Na) โดยการคัดเลือกและจัดลำดับสัญลักษณ์ที่มีความยาวมากกว่า และใช้ Word Boundary RegExp สำหรับสัญลักษณ์ภาษาอังกฤษเพื่อป้องกันปัญหานี้อย่างถาวร
  - **ระบบคำนวณสูตรเตรียมสารเคมีอัตโนมัติ (Dynamic Solution Prep Guide):** เพิ่มฟีเจอร์คำนวณและแสดงขั้นตอนการเตรียมสารละลาย (เช่น ความเข้มข้น M และปริมาตร mL/L) สำหรับสารเคมีทุกประเภทได้แบบไดนามิก โดยแชทบอตจะสกัดสูตรเคมี (เช่น NaOH, NaCl, HCl, H2SO4, ฯลฯ) ประเมินมวลโมเลกุล (Molar Mass) อัตโนมัติ และแสดงวิธีคำนวณพร้อมขั้นตอนการเตรียมทีละขั้นตอนสำหรับของแข็ง (Solid) และของเหลว (Liquid dilution) สต็อกกรดเข้มข้นอย่างเหมาะสมและมีความปลอดภัยสูง
- **ปรับปรุงระบบส่งออก (Export CSV/PDF) สำหรับรายการสั่งซื้อสะสม:**
  - **รายละเอียดส่วนลด:** นำข้อมูลคำนวณรายละเอียดส่วนลด (รวมก่อนลดราคา, ส่วนลดสะสม, และสุทธิหลังหักส่วนลด) ใส่ในไฟล์ส่งออกทั้งรูปแบบ CSV และ PDF
  - **หัวตารางคงที่สำหรับ PDF:** เพิ่ม CSS `thead { display: table-header-group; }` และควบคุม page-break เพื่อให้หัวตารางทำซ้ำอัตโนมัติที่ส่วนบนสุดเมื่อพิมพ์หรือบันทึก PDF หลายหน้า

  - แสดงหน้าป๊อปอัปคำแนะนำการจัดเก็บตามกลุ่มที่ถูกต้องและแนะนำการแยกห่างสารเคมีอันตรายอย่างครบถ้วน

## [1.7.0] - 2026-06-17
### Added
- **ระบบผู้ช่วยห้องแล็บอัจฉริยะ (AI Assistant Catalyst):**
  - ติดตั้งกล่องแชทบอท **"แคททาไลต์ (Catalyst)"** ที่มุมล่างขวาของหน้าจอ (ปุ่ม FAB รูปหุ่นยนต์ขนาดเล็ก 44px)
  - ความสามารถแชทบอท: ค้นหาพัสดุ/พิกัดจัดเก็บแบบ Real-time, ตรวจสอบสินค้าสต็อกต่ำ, ตรวจสอบสารหมดอายุ และคู่มือขั้นตอนการใช้งาน
- **ระบบฐานความรู้เคมีจำลอง (Chemical Knowledge Base):**
  - ค้นหาคำอ่านภาษาไทยสะกดออกเสียงสำหรับสารเคมีพื้นฐาน (เช่น กรดไฮโดรคลอริก, โซเดียมไฮดรอกไซด์, เอทานอล ฯลฯ)
  - รายงานผลผลิตของปฏิกิริยาเคมีและสมการเคมีที่ดุลแล้วเมื่อนำสารสองชนิดมาผสมกัน
  - ระบบตรวจจับและรายงานรหัสสารเคมีในคลังหากอยู่นอกฐานข้อมูลแชทบอท
- **ระบบส่งออกประวัติหลังบ้าน (Advanced History Exporters):**
  - ปุ่มส่งออกข้อมูล **CSV** และ **PDF** สำหรับตาราง **"ประวัติการยืม-คืนพัสดุ"** และ **"ประวัติการใช้ห้องปฏิบัติการ"**
  - รองรับการแสดงผลภาษาไทยสมบูรณ์แบบบน Microsoft Excel (UTF-8 BOM)
  - ระบบพิมพ์ PDF ออกใบรายงานรูปแบบทางการ สะอาด สวยงาม
  - จำกัดสิทธิ์การแสดงปุ่มส่งออกเฉพาะผู้ดูแลระบบและอาจารย์ที่เข้าสู่ระบบหลังบ้านสำเร็จเท่านั้น

### Changed
- ปรับปรุงให้กำหนดส่งคืนพัสดุสะสมเป็น **7 วันโดยอัตโนมัติ** นับจากวันที่ใช้งาน (วันที่ทำรายการยืม) แทนของเดิม
- ลบเอฟเฟกต์การเบลอฉากหลัง (Backdrop Blur) ขณะเปิดใช้งานแชทบอทออก เพื่อไม่ให้บดบังการแสดงผลข้อมูลหน้ารายการหลัก
- ปรับขนาดปุ่มเปิดแชทบอท FAB ให้กระทัดรัดขึ้นเป็น `44px` พร้อมไอคอนรูปหุ่นยนต์บอทขนาด `18px`
