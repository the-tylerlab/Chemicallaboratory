# Changelog (บันทึกการปรับปรุงระบบ)

บันทึกประวัติการพัฒนาและการอัปเกรดระบบจัดการห้องปฏิบัติการเคมีและอุปกรณ์วิทยาศาสตร์ (Chemical Laboratory & Scientific Equipment System)

---

## [1.7.1] - 2026-06-18
### Added
- **ระบบแสดงผลรวมสถิติพัสดุ (Total Sum Display):**
  - แสดงป้าย Badge จำนวนรวมทั้งหมดของรายการพัสดุเป็นภาษาไทย (เช่น `X รายการ`) ตรงส่วนหัวของการ์ด "สัดส่วนหมวดหมู่พัสดุ" บนแดชบอร์ด

### Fixed
- **แก้ไขหมวดหมู่พัสดุในการ์ดสัดส่วน (Category Breakdown Alignment):**
  - ปรับปรุงการคำนวณสัดส่วนหมวดหมู่ให้แมปตามประเภทจริง ได้แก่ `สารเคมี`, `อุปกรณ์วิทยาศาสตร์`, `เครื่องแก้ว`, และ `วัสดุสิ้นเปลือง` (และซ่อนหมวดหมู่ `อื่นๆ` หากไม่มีพัสดุ) ป้องกันการสับสนของข้อมูลและลดปัญหาข้อมูลไปกองอยู่ใต้หมวดหมู่ อื่นๆ

---

## [1.7.0] - 2026-06-17
### Added
- **ระบบผู้ช่วยห้องแล็บอัจฉริยะ (AI Assistant Catalyst):**
  - ติดตั้งกล่องแชทบอท **"แคททาไลต์ (Catalyst)"** ที่มุมล่างขวาของหน้าจอ (ปุ่ม FAB รูปหุ่นยนต์ขนาดเล็ก 44px)
  - ความสามารถแชทบอท: ค้นหาพัสดุ/พิกัดจัดเก็บแบบ Real-time, ตรวจสอบสินค้าสต็อกต่ำ, ตรวจสอบสารหมดอายุ และคู่มือขั้นตอนการใช้งาน
- **ระบบฐานความรู้เคมีจำลอง (Chemical Knowledge Base):**
  - ค้นหาคำอ่านภาษาไทยสะกดออกเสียงสำหรับสารเคมีพื้นฐาน (เช่น กรดไฮโดรคลอริก, โซเดียมไฮดรอกไซด์, เอทานอล ฯลฯ)
  - รายงานผลผลิตของปฏิกิริยาเคมีและสมการเคมีที่ดุลแล้วเมื่อนำสารสองชนิดมาผสมกัน
  - ระบบตรวจจับและรายงานรหัสสารเคมีในคลังหากอยู่นอกฐานข้อมูลแชทบอท
- **ระบบส่งออกประวัติหลังบ้าน (Advanced History Exporters):**
  - ปุ่มส่งออกข้อมูล **CSV** และ **PDF** สำหรับตาราง **"ประวัติการยืม-คืนพัสดุ"** และ **"ประวัติการใช้ห้องปฏิบัติการ"**
  - รองรับการแสดงผลภาษาไทยสมบูรณ์แบบบน Microsoft Excel (UTF-8 BOM)
  - ระบบพิมพ์ PDF ออกใบรายงานรูปแบบทางการ สะอาด สวยงาม
  - จำกัดสิทธิ์การแสดงปุ่มส่งออกเฉพาะผู้ดูแลระบบและอาจารย์ที่เข้าสู่ระบบหลังบ้านสำเร็จเท่านั้น

### Changed
- ปรับปรุงให้กำหนดส่งคืนพัสดุสะสมเป็น **7 วันโดยอัตโนมัติ** นับจากวันที่ใช้งาน (วันที่ทำรายการยืม) แทนของเดิม
- ลบเอฟเฟกต์การเบลอฉากหลัง (Backdrop Blur) ขณะเปิดใช้งานแชทบอทออก เพื่อไม่ให้บดบังการแสดงผลข้อมูลหน้ารายการหลัก
- ปรับขนาดปุ่มเปิดแชทบอท FAB ให้กระทัดรัดขึ้นเป็น `44px` พร้อมไอคอนรูปหุ่นยนต์บอทขนาด `18px`

## [1.6.0] - 2026-05-28
### Added
- **ระบบจองห้องปฏิบัติการ (Lab Booking System):**
  - เมนูจองห้องปฏิบัติการ (Lab 1, Lab 2, Lab 3)
  - ระบบเลือกคาบเรียนและวัตถุประสงค์ พร้อมตารางบันทึกการเข้าใช้ห้องปฏิบัติการ
- **คุณสมบัติความปลอดภัยของสารเคมี (Safety Feature Suite):**
  - เพิ่มการจัดเก็บประเภทอันตรายสารเคมี (Chemical Danger Type)
  - เพิ่มตราสัญลักษณ์ความเป็นอันตรายตามระบบสากล (GHS Hazard Pictograms)
  - เพิ่มฟิลด์เชื่อมต่อ SDS Link (Safety Data Sheet)
- **ระบบจัดการพัสดุชำรุด/ส่งซ่อม (Damaged & Repair Status Tracking):**
  - ระบบสรุปจำนวนพัสดุเสียหาย แยกออกจากการจัดเก็บปกติ
  - เจ้าหน้าที่สามารถจัดการสถานะส่งซ่อม ซ่อมสำเร็จ หรือจำหน่ายทิ้ง (ตัดสต็อก) ได้ผ่านโมดอลประแจ
- **ระบบสลับบทบาทจำลอง (Role Switcher UI):**
  - เพิ่มแถบเลือกสลับสถานะจำลอง (นักเรียน / อาจารย์ / ผู้ดูแลระบบ) บริเวณหน้าควบคุมหลัก

## [1.5.0] - 2026-05-15
### Added
- **การเชื่อมโยงระบบฐานข้อมูล Cloud (Firebase Firestore):**
  - รองรับการบันทึก อ่าน และอัปเดตข้อมูลแบบ Real-time ร่วมกับ Firebase
  - เพิ่มระบบตรวจเช็คสถานะการเชื่อมต่อ (Online/Offline Status Indicator)
- **ระบบสแกนบาร์โค้ด (Barcode & Camera Scanner):**
  - ค้นหาพัสดุและสารเคมีผ่านช่องค้นหาบาร์โค้ด หรือเปิดกล้องเว็บแคมเพื่อสแกนรหัสแท่งพัสดุโดยตรง
- **สิทธิเข้าถึงข้อมูลเบื้องต้น:**
  - แบ่งส่วนการแสดงผลข้อมูลแบบเปิดอ่านทั่วไป (Viewer) และโหมดลงทะเบียนเพื่อแก้ไขสต็อกสินค้า (Staff/Admin)

---

## [1.0.0] - 2026-04-10
### Added
- **ระบบคลังสารเคมีพื้นฐาน (Basic Inventory System):**
  - บันทึกรหัสสินค้า, ชื่อเคมี/อุปกรณ์, หมวดหมู่, จำนวนคงคลัง, หน่วยนับ
  - บันทึกสถานที่จัดเก็บ (ตู้เก็บของ, ชั้นวาง) และวันหมดอายุสารเคมี
  - ระบบแสดงผลตารางรายการพัสดุทั้งหมด พร้อมระบบค้นหาอักษรและการจำแนกหมวดหมู่
  - ระบบยืม-คืนพัสดุพื้นฐานแบบหักลบจำนวนทันที
